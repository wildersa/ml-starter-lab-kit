# Overfitting

Overfitting happens when the model memorizes training data instead of learning a useful pattern.

## Signs

- great training score;
- weak validation/test score;
- unstable performance across splits;
- model depends on noisy features.

## Practical fixes

- simplify the model;
- use validation data;
- reduce noisy features;
- regularize when applicable;
- collect more data when possible.
