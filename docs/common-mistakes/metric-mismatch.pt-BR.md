# Métrica incompatível

Métrica incompatível acontece quando a métrica não combina com a decisão real.

## Exemplos

- usar accuracy em classificação muito desbalanceada;
- otimizar cliques quando o negócio se importa com conversão;
- usar MAPE quando valores reais podem ser zero;
- reportar só score global quando comportamento por segmento importa.

## Regra prática

Escolha a métrica que melhor representa o custo de errar.
