# Ignorar o baseline

Baseline é o resultado simples que seu modelo precisa superar.

## Exemplos

- classe majoritária;
- valor médio;
- último valor conhecido;
- política aleatória;
- regra simples de negócio.

## Por que importa

Sem baseline, você não sabe se o modelo é útil.

Um modelo complexo que quase não supera uma regra simples talvez não compense a complexidade.
