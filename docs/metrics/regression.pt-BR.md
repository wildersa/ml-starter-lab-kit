# Métricas de regressão

Use métricas de regressão quando o modelo prevê um número.

## MAE

Erro absoluto médio.

Fácil de explicar: erro médio em valor absoluto.

## RMSE

Raiz do erro quadrático médio.

Penaliza erros grandes mais que o MAE.

## R²

Mostra quanta variância o modelo explica.

É útil, mas não basta sozinho.

## MAPE

Erro percentual absoluto médio.

É fácil de comunicar, mas problemático quando valores reais estão perto de zero.
