# Classification Metrics

Use classification metrics when the model predicts a class.

## Accuracy

Fraction of correct predictions.

Good for balanced datasets. Risky when classes are imbalanced.

## Precision

Of everything predicted as positive, how much was really positive.

Useful when false positives are expensive.

## Recall

Of everything truly positive, how much the model found.

Useful when false negatives are expensive.

## F1-score

Balance between precision and recall.

Useful for imbalanced classification.

## AUC

Measures ranking quality across thresholds.

Useful when the model outputs probabilities.
