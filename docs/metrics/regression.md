# Regression Metrics

Use regression metrics when the model predicts a number.

## MAE

Mean Absolute Error.

Easy to explain: average absolute error.

## RMSE

Root Mean Squared Error.

Penalizes large errors more than MAE.

## R²

Shows how much variance is explained by the model.

Useful, but not enough alone.

## MAPE

Mean Absolute Percentage Error.

Easy to communicate, but problematic when actual values are close to zero.
