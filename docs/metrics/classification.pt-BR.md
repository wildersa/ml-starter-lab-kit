# Métricas de classificação

Use métricas de classificação quando o modelo prevê uma classe.

## Accuracy

Percentual de previsões corretas.

Boa para bases balanceadas. Perigosa quando as classes são desbalanceadas.

## Precision

De tudo que foi previsto como positivo, quanto era realmente positivo.

Útil quando falso positivo é caro.

## Recall

De tudo que era realmente positivo, quanto o modelo encontrou.

Útil quando falso negativo é caro.

## F1-score

Equilíbrio entre precision e recall.

Útil para classificação desbalanceada.

## AUC

Mede qualidade de ranking em vários thresholds.

Útil quando o modelo gera probabilidades.
