# Before Evaluation

Use this checklist before trusting model results.

- [ ] The model was not evaluated only on training data.
- [ ] The validation/test split was not leaked.
- [ ] The metric matches the problem.
- [ ] The baseline result is available.
- [ ] Feature impact was checked for important new features.
- [ ] Results were inspected by segment when relevant.
- [ ] Errors were reviewed with examples.
- [ ] Suspiciously high scores were investigated.
- [ ] Random seed or reproducibility notes are documented.
- [ ] Limitations are written down.
