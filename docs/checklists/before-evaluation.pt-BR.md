# Antes da avaliação

Use este checklist antes de confiar nos resultados do modelo.

- [ ] O modelo não foi avaliado apenas nos dados de treino.
- [ ] O split de validação/teste não sofreu vazamento.
- [ ] A métrica combina com o problema.
- [ ] O resultado do baseline está disponível.
- [ ] O impacto de features importantes foi checado.
- [ ] Resultados foram inspecionados por segmento, quando relevante.
- [ ] Erros foram revisados com exemplos.
- [ ] Scores suspeitamente altos foram investigados.
- [ ] Seed aleatória ou notas de reprodutibilidade estão documentadas.
- [ ] Limitações foram escritas.
