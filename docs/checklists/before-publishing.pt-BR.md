# Antes de publicar

Use este checklist antes de compartilhar notebook, relatório ou repositório.

- [ ] O README explica o problema e como executar o projeto.
- [ ] Nenhum segredo, token ou dado privado foi commitado.
- [ ] Fonte e licença do dataset estão documentadas.
- [ ] Artefatos grandes não foram versionados por acidente.
- [ ] Limitações do modelo estão documentadas.
- [ ] Métricas estão explicadas.
- [ ] Relatórios gerados estão em `reports/`.
- [ ] Notebooks são legíveis de cima para baixo.
- [ ] Lógica reutilizável foi movida para `src/`.
- [ ] O projeto pode ser entendido sem explicação oral.
