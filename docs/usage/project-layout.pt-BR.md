# Layout do Starter e do Projeto Gerado

Mantenha o starter kit em sua própria pasta.

Crie projetos de Machine Learning gerados em uma pasta separada.

Layout recomendado:

```text
workspace/
├── ml-starter-lab-kit/
└── meu-projeto-ml/
```

Isso mantém o starter reutilizável e evita que arquivos gerados se misturem com o repositório do starter.
