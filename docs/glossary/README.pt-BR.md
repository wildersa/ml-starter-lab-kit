# Glossário de ML

Definições curtas para termos comuns de Machine Learning.

| Termo | Significado |
|---|---|
| Feature | coluna de entrada usada por um modelo |
| Target | valor que o modelo tenta prever |
| Baseline | referência simples que o modelo precisa superar |
| Split | separação entre treino, validação e teste |
| Data leakage | uso de informação indisponível no momento da previsão |
| Overfitting | quando o modelo memoriza demais os dados de treino |
| Underfitting | quando o modelo é simples demais para aprender o padrão |
| Métrica | número usado para avaliar qualidade do modelo |
| Hiperparâmetro | configuração escolhida antes ou ao redor do treino |
| Inferência | uso de um modelo treinado para fazer predições |
| Fine-tuning | adaptação de um modelo pré-treinado para uma tarefa específica |
| Feature engineering | criação de colunas úteis de entrada |
| Pré-processamento | preparação dos dados brutos antes da modelagem |
| Artefato | saída salva, como modelo, métricas ou relatório |
