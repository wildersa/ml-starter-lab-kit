# Glossário de ML

Definições curtas para termos comuns de Machine Learning.

| Termo | Significado | Guia relacionado |
|---|---|---|
| Feature | coluna de entrada usada por um modelo | [Feature engineering](../models/feature-engineering.pt-BR.md) |
| Target | valor que o modelo tenta prever | [Modelos supervisionados](../models/supervised.pt-BR.md) |
| Baseline | referência simples que o modelo precisa superar | [Ignorar baseline](../common-mistakes/ignoring-baseline.pt-BR.md) |
| Split | separação entre treino, validação e teste | [Split ruim](../common-mistakes/bad-split.pt-BR.md) |
| Data leakage | uso de informação indisponível no momento da previsão | [Data leakage](../common-mistakes/data-leakage.pt-BR.md) |
| Overfitting | quando o modelo memoriza demais os dados de treino | [Overfitting](../common-mistakes/overfitting.pt-BR.md) |
| Underfitting | quando o modelo é simples demais para aprender o padrão | [Checklist antes da avaliação](../checklists/before-evaluation.pt-BR.md) |
| Métrica | número usado para avaliar qualidade do modelo | [Visão geral de métricas](../metrics/README.pt-BR.md) |
| Hiperparâmetro | configuração escolhida antes ou ao redor do treino | [Hiperparâmetros](../workflows/hyperparameters.pt-BR.md) |
| Inferência | uso de um modelo treinado para fazer predições | [Camadas de pipeline ML](../architectures/layers.pt-BR.md) |
| Fine-tuning | adaptação de um modelo pré-treinado para uma tarefa específica | [Fine-tuning](../workflows/fine-tuning.pt-BR.md) |
| Feature engineering | criação de colunas úteis de entrada | [Feature engineering](../models/feature-engineering.pt-BR.md) |
| Pré-processamento | preparação dos dados brutos antes da modelagem | [Pré-processamento](../workflows/preprocessing.pt-BR.md) |
| Artefato | saída salva, como modelo, métricas ou relatório | [Ciclo básico de MLOps](../architectures/mlops-lifecycle.pt-BR.md) |
