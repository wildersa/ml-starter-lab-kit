# ML Glossary

Short definitions for common Machine Learning terms.

| Term | Meaning |
|---|---|
| Feature | input column used by a model |
| Target | value the model tries to predict |
| Baseline | simple reference result to beat |
| Split | separation between train, validation, and test data |
| Data leakage | use of information unavailable at prediction time |
| Overfitting | when the model memorizes training data too much |
| Underfitting | when the model is too simple to learn the pattern |
| Metric | number used to evaluate model quality |
| Hyperparameter | setting chosen before or around training |
| Inference | using a trained model to make predictions |
| Fine-tuning | adapting a pretrained model to a specific task |
| Feature engineering | creating useful input columns |
| Preprocessing | preparing raw data before modeling |
| Artifact | saved output such as model, metrics, or report |
