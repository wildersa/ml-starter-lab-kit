# ML Glossary

Short definitions for common Machine Learning terms.

| Term | Meaning | Related guide |
|---|---|---|
| Feature | input column used by a model | [Feature engineering](../models/feature-engineering.md) |
| Target | value the model tries to predict | [Supervised learning](../models/supervised.md) |
| Baseline | simple reference result to beat | [Ignoring baseline](../common-mistakes/ignoring-baseline.md) |
| Split | separation between train, validation, and test data | [Bad split](../common-mistakes/bad-split.md) |
| Data leakage | use of information unavailable at prediction time | [Data leakage](../common-mistakes/data-leakage.md) |
| Overfitting | when the model memorizes training data too much | [Overfitting](../common-mistakes/overfitting.md) |
| Underfitting | when the model is too simple to learn the pattern | [Before evaluation](../checklists/before-evaluation.md) |
| Metric | number used to evaluate model quality | [Metrics overview](../metrics/README.md) |
| Hyperparameter | setting chosen before or around training | [Hyperparameters](../workflows/hyperparameters.md) |
| Inference | using a trained model to make predictions | [ML pipeline layers](../architectures/layers.md) |
| Fine-tuning | adapting a pretrained model to a specific task | [Fine-tuning](../workflows/fine-tuning.md) |
| Feature engineering | creating useful input columns | [Feature engineering](../models/feature-engineering.md) |
| Preprocessing | preparing raw data before modeling | [Preprocessing](../workflows/preprocessing.md) |
| Artifact | saved output such as model, metrics, or report | [MLOps lifecycle](../architectures/mlops-lifecycle.md) |
